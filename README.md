## 📌 Project Overview

This project explores Netflix data to uncover trends in content, genres, and release patterns.

---

## 🛠️ Tools & Technologies

- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset

- Source: Netflix dataset (movies & TV shows)
- Contains:
  - Title  
  - Genre  
  - Release year  
  - Country  
  - Type (Movie/TV Show)  

---

## 📊 Key Analysis

- 📈 Content growth over years  
- 🎭 Most popular genres  
- 🌍 Country-wise distribution  
- 🎬 Movies vs TV Shows
